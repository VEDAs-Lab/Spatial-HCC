# Spatial HCC — Spatial Immune Coupling Index (SICI)

Analysis code and output figure(s) for the Spatial HCC project, which characterises immune-tumour spatial coupling in hepatocellular carcinoma (HCC) using Visium spatial transcriptomics and scRNA-seq deconvolution.

## Data Access

The `Data/` folder is not included in this repository due to file size. Download the required datasets as described below, and place them in a `Data/` directory at the root of this repo.

### 1. Spatial Transcriptomics — GSE238264 (10x Visium, HCC)

**Source:** NCBI GEO — [GSE238264](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE238264)

This dataset contains 7 HCC Visium samples (4 responders, 3 non-responders).

```bash
# Download via GEO (requires SRA tools or direct download from GEO)
# Sample accessions: GSM7661255–GSM7661261
wget "https://www.ncbi.nlm.nih.gov/geo/download/?acc=GSE238264&format=file" \
     -O Data/GSE238264_RAW.tar
tar -xf Data/GSE238264_RAW.tar -C Data/GSE238264_RAW/
```

Expected structure after extraction:
```
Data/GSE238264_RAW/
├── HCC1R/   (GSM7661255)
├── HCC2R/   (GSM7661256)
├── HCC3R/   (GSM7661257)
├── HCC4R/   (GSM7661258)
├── HCC5NR/  (GSM7661259)
├── HCC6NR/  (GSM7661260)
└── HCC7NR/  (GSM7661261)
```

Each sample folder should contain `filtered_feature_bc_matrix/`, `spatial/`, and `filtered_feature_bc_matrix.h5`.

### 2. scRNA-seq Reference — GSE149614 (HCC scRNA-seq)

**Source:** NCBI GEO — [GSE149614](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE149614)

```bash
# Count matrix (~3.5 GB uncompressed)
wget "https://www.ncbi.nlm.nih.gov/geo/download/?acc=GSE149614&format=file&file=GSE149614_HCC.scRNAseq.S71915.count.txt.gz" \
     -O Data/GSE149614_HCC.scRNAseq.S71915.count.txt.gz

# Metadata
wget "https://www.ncbi.nlm.nih.gov/geo/download/?acc=GSE149614&format=file&file=GSE149614_HCC.metadata.updated.txt.gz" \
     -O Data/GSE149614_HCC.metadata.updated.txt.gz

# Decompress
gunzip Data/GSE149614_HCC.scRNAseq.S71915.count.txt.gz
gunzip Data/GSE149614_HCC.metadata.updated.txt.gz
```

### 3. TCGA LIHC Survival Data

**Source:** UCSC Xena Browser — [TCGA Liver Cancer (LIHC)](https://xenabrowser.net/datapages/?cohort=TCGA%20Liver%20Cancer%20(LIHC))

Download the following two files and place them in `Data/`:

| File | Description | Xena Dataset |
|---|---|---|
| `TCGA.LIHC.sampleMap_HiSeqV2` | Gene expression (log2 RSEM) | `TCGA.LIHC.sampleMap/HiSeq V2` |
| `survival_LIHC_survival.txt` | Clinical survival data | `TCGA.LIHC.sampleMap/LIHC_survival` |

Direct download links (via UCSC Xena):
```bash
wget "https://tcga-xena-hub.s3.us-east-1.amazonaws.com/download/TCGA.LIHC.sampleMap%2FHiSeqV2.gz" \
     -O Data/TCGA.LIHC.sampleMap_HiSeqV2.gz && gunzip Data/TCGA.LIHC.sampleMap_HiSeqV2.gz

wget "https://tcga-xena-hub.s3.us-east-1.amazonaws.com/download/survival%2FLIHC_survival.txt" \
     -O Data/survival_LIHC_survival.txt
```

## Dependencies

A `requirements.txt` or `environment.yml` should be added. Key packages used:

- `scanpy`, `anndata` — scRNA-seq and AnnData processing
- `squidpy` — spatial transcriptomics analysis
- `cell2location` — spatial deconvolution
- `lifelines` — survival analysis
- `numpy`, `pandas`, `scipy`, `scikit-learn`
- `matplotlib`, `seaborn`
